### 整数
* 定义 a = 125
* Python 可以处理 “任意” 大小的整数，包括负数。
* 支持二进制，八进制，十进制，十六进制

前缀|例子|进制
-|-|-
0b或者0B|a=0b1010|2
0o或者0O|a=0b12|8
无|a=10|10
0x或者0X|a=0xa|16
